# implement-screen.md

以下の条件で 1 画面を実装してください。

- 正本は `.design/design.pen`
- レイアウト、余白、情報階層、色、主従関係を忠実に再現すること
- 勝手な再設計は禁止
- Focus は青系、Break は緑系を守ること
- React は表示層中心、タイマー主制御は Rust / Tauri 側に寄せること
- 重い依存は追加しないこと

作業前に以下を短く示してください。

- 変更対象
- 実装方針
- `.design/design.pen` との対応
- パフォーマンス影響
- 依存追加の有無

作業後は以下を報告してください。

- 実装内容
- 正本との差分がある場合の理由
- テスト / 確認内容
- 残課題
