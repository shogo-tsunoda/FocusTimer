# Design Implementation Skill

## 目的
Pencil で作成した `.pen` ファイルとエクスポート画像をもとに、
UI を忠実に React / CSS へ落とし込むためのスキル。

---

## 原則
- Pencil を正本とする
- 再設計しない
- 余白、階層、色、主従関係を守る
- 主役はタイマー表示

---

## 実装観点
- Focus は青系
- Break は緑系
- タイマー中央の数字を最重要視
- 補助ラベルは控えめ
- セッション進行の意味を明確にする
- ボタン意味を誤解させない

---

## 注意点
- 「一般的なUI」に寄せすぎない
- 勝手にカード構造やナビゲーションを足さない
- 影、角丸、余白の密度を変えすぎない

---

## 出力ルール
UI変更提案が必要なら以下を示す。

- 元デザインとの差分
- 必要理由
- 影響範囲
