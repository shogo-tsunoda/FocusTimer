# release.md

## Windows / macOS リリース前チェック

- 両 OS でビルドできること
- 起動、終了、再起動が安定していること
- 通知、トレイ、ウィンドウ制御が自然に動くこと
- 初回起動時の設定読み込みが破綻しないこと

## UI 確認

- `.design/design.pen` との差分が説明可能であること
- Focus は青系、Break は緑系で一貫していること
- タイマー表示が主役として視認しやすいこと
- 余白、整列、テキストの見切れがないこと
- hover / active / disabled が破綻していないこと

## パフォーマンス確認

- idle 時 CPU 使用率が低いこと
- 常駐時メモリが過剰でないこと
- 起動時間が悪化していないこと
- 毎フレーム更新や不要なポーリングが入っていないこと

## 主要フロー確認

- start / pause / resume / reset / skip
- Focus から Break への遷移
- Long Break への遷移
- 設定変更、保存、再起動後の復元
- 通知、トレイ、ウィンドウ操作

## リリース禁止条件

- `.design/design.pen` と明確に乖離している
- タイマー精度や状態遷移に不安がある
- 型、lint、clippy、主要テストが未通過
- OS 片方で主要フローが未確認
- 依存追加や大きな変更の理由が説明できない
